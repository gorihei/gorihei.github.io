## コードブロック

フィボちゃん。

```js [fibo.js]
// 典型的なフィボナッチ（遅い版）
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
  // O(2^n)で激重（あるある）
}

// 改良版（メモ化）
function fibonacciOptimized(n, memo = {}) {
  if (n in memo) return memo[n];
  if (n <= 1) return n;

  memo[n] = fibonacciOptimized(n - 1, memo) + fibonacciOptimized(n - 2, memo);
  return memo[n];
}
```
